my first blog

whh 发表于 2021-07-18

时间如白驹过隙，尚未察觉2021半年已过，年初的目标仍然远在终点，大部分期许仍未完成。

工作

  1. 这半年的工作仍然是普普通通，项目运维和需求开发。function，es能力是有所进步，但是总体没有大的突破。
  • 程序员的硬实力，算法，源码阅读，设计模式没有提升
  • 学习不论读书还是视频，总是浅尝辄止，好像什么都知道，但是什么都说不上来。

生活

  • 吃喝玩乐不错过，理财进行,无理论学习，随心而行，财务管理错乱无序
  • 一直存在于嘴边的远方，始终没有出发，这个不取决于兜里有没有钱
  • 一直没有锻炼，懒于运动，以为谈恋爱就可以有动力运动，其实一切的原动力还是属于自己，而非外务
  • 早睡早起，一直没有实现

学习

总的来说，只是期许

  • 理财观念需加强，有学习，有存款
  • 每日刷题，2道题，重要的是坚持
  • java 基础夯实
  • 框架源码熟悉
  • 设计模式
  • DDD 领域驱动设计概念熟悉
  • 微服务学习
